# 健康报告上传功能使用指南

## 功能概述

AI健康报告解读与持续管理助手现已支持完整的体检报告上传与解读功能。用户可以通过拖拽或点击的方式上传PDF格式的体检报告，系统会自动进行AI解读并生成个性化健康建议。

## 主要特性

### 1. 多种上传方式
- **拖拽上传**：直接将PDF文件拖拽到上传区域
- **点击上传**：点击上传区域选择文件
- **批量上传**：支持一次上传最多5个文件

### 2. 智能进度展示
上传过程分为4个阶段，实时展示：
- 📤 **文件上传**（0-30%）：上传文件到系统
- 🔍 **OCR识别**（30-60%）：AI识别报告内容
- 📊 **数据提取**（60-80%）：提取健康指标数据
- 🧠 **AI分析**（80-100%）：生成个性化健康建议

### 3. 家庭成员管理
- 可为不同家庭成员分别上传报告
- 报告自动关联到对应成员
- 支持查看每个成员的历史报告

### 4. 上传状态管理
- ✅ **已解读**：报告已成功解读并生成建议
- 🔄 **解读中**：正在进行AI分析
- ⏳ **待审核**：需要人工审核
- ❌ **失败**：上传失败，可重试

## 使用场景

### 场景1：首次使用
1. 访问首页，系统自动显示欢迎引导页
2. 点击"上传首份体检报告"按钮
3. 选择或拖拽PDF报告文件
4. 等待AI自动解读（约30-60秒）
5. 解读完成后自动跳转到健康档案页面

### 场景2：添加新报告
1. 在首页点击"上传报告"按钮
2. 选择要上传的家庭成员
3. 上传新的体检报告
4. 查看更新后的健康建议

### 场景3：健康档案页面上传
1. 访问"健康档案"页面
2. 点击顶部"上传报告"按钮
3. 在弹窗中选择成员和文件
4. 上传完成后自动刷新档案数据

## 文件要求

- **格式**：仅支持PDF格式
- **大小**：单个文件不超过50MB
- **数量**：单次最多上传5个文件
- **内容**：建议上传正规医疗机构出具的体检报告

## 功能位置

### 1. 首页仪表盘
- **路径**：`/` 或 `/dashboard`
- **入口**：页面顶部"上传报告"按钮
- **特点**：首次使用时显示大型引导卡片

### 2. 健康档案页
- **路径**：`/profiles`
- **入口**：页面顶部"上传报告"按钮（弹窗形式）
- **特点**：可查看历史报告列表

## 上传后的数据

### 自动生成的内容
1. **健康指标提取**：
   - 血糖、血脂、尿酸等关键指标
   - 指标趋势分析
   - 异常值标记

2. **AI个性化建议**：
   - 🥗 饮食调理方案
   - 💊 维生素与补剂建议
   - 💉 药物治疗建议（仅供参考）
   - 💪 运动健身方案

3. **数据存储**：
   - 所有数据仅保存在浏览器本地
   - 不会上传到任何服务器
   - 确保数据隐私安全

### 本地存储
- 使用 `localStorage` 存储所有数据
- 键名：`health_reports`
- 支持跨会话数据持久化

## 注意事项

### 隐私保护
- ⚠️ 本应用不收集任何个人健康信息
- ⚠️ 所有数据仅保存在用户本地浏览器
- ⚠️ 清除浏览器缓存会删除所有数据

### 医疗合规
- ⚠️ AI建议仅供健康管理参考
- ⚠️ 不作为医疗诊断依据
- ⚠️ 具体治疗方案请咨询专业医师
- ⚠️ 严禁自行用药

### 浏览器兼容性
- 推荐使用 Chrome、Edge、Firefox、Safari 最新版本
- 需要启用 JavaScript 和 localStorage
