# AI健康报告解读与持续管理助手

一个面向个人及家庭用户的非诊断型AI健康管理协同工具，聚焦体检报告深度解读、多维健康建议生成、长期指标追踪与可信赖的AI对话交互。

> **核心价值主张**："一次上传，持续守护" —— 让每份体检报告成为长期健康旅程的起点，而非一次性结论。

## 📋 项目简介

### 🎯 核心定位
- **产品类型**：健康信息参考工具（非医疗诊断服务）
- **目标用户**：关注健康的成年人（25-65岁）、慢性病风险人群、家庭健康管理者
- **核心功能**：体检报告AI解读 + 健康档案管理 + 智能对话咨询 + 指标趋势追踪

### ✨ 核心功能模块

#### 1️⃣ PDF报告上传与AI解析
- 📄 支持拖拽上传，单次最多5份PDF
- 🤖 AI智能解析（OCR + 语义理解）
- 📊 实时处理进度展示（语义解析 → 异常识别 → 指南匹配 → 建议生成）
- ✅ 解析完成自动生成健康档案

#### 2️⃣ 健康档案与多成员管理
- 👨‍👩‍👧‍👦 支持家庭多成员档案管理
- 📈 关键指标卡片展示（空腹血糖、尿酸、血脂等）
- 📉 趋势图表可视化（近1年/2年数据对比）
- ⚠️ 状态分级：正常/警示/危险（颜色编码）

#### 3️⃣ AI对话中心（记忆驱动）
- 💬 上下文感知的智能对话
- 🧠 记住历史报告和对话内容
- 📚 所有建议附带权威来源（指南名称+年份）
- 🎯 快捷指令：总结报告、生成饮食计划、复查提醒

#### 4️⃣ 健康概览仪表盘
- 🎯 今日关注：最需干预的指标
- 📊 趋势洞察：近30天指标变化热力图
- ✅ AI建议进度：已执行建议数/总建议数

#### 5️⃣ 设置与合规中心
- 🔒 隐私模糊模式：数值显示为区间
- 💾 数据导出/清除功能
- ⚠️ 完整的医疗免责声明
- 📖 权威指南来源说明

## 🚀 技术栈

- **前端框架**：React 18 + TypeScript
- **路由**：React Router 7
- **UI组件**：Radix UI + Tailwind CSS v4
- **图表库**：Recharts
- **图标**：Lucide React
- **动画**：Motion (Framer Motion)
- **构建工具**：Vite 6

## 🎯 核心特性

### ✅ 严格医疗合规
- 全局合规提示横幅（固定顶部，不可隐藏）
- 所有AI建议末尾自动附加免责声明
- 明确标注"不用于诊断、治疗、用药决策"
- 禁用医疗行为动词（确诊、治疗、处方）

### ✅ 记忆驱动型AI
- 记忆锚点横幅：显示当前上下文（报告+关注指标+上次对话时间）
- 跨报告对比分析（2023 vs 2024）
- 支持@提及指标（如"@空腹血糖"）

### ✅ 权威来源标注
所有AI建议基于公开权威指南：
- 《中国居民膳食指南2022》
- 《中国成人血脂异常防治指南》
- 《中国高尿酸血症与痛风诊疗指南》
- AHA（美国心脏协会）指南
- KDIGO慢性肾脏病管理指南

### ✅ 数据隐私保护
- 所有数据存储在浏览器本地（LocalStorage）
- 不上传到任何服务器
- 可随时导出或清除数据
- 隐私模糊模式：数值显示为区间

## 🎨 设计规范

- **主色系**：#2563EB（科技蓝）、#10B981（健康绿）、#F59E0B（警示橙）、#EF4444（危险红）
- **字体**：Inter（西文）+ HarmonyOS Sans SC（中文）
- **字号**：正文14px，标题16-18px
- **响应式布局**：支持桌面（≥1280px）、平板（768-1279px）、手机（≤767px）

## 📝 使用说明

### 快速开始
1. **上传报告**：点击"我的报告" → 拖拽或选择PDF体检报告
2. **等待解析**：AI处理约2-3分钟（实时显示进度）
3. **查看结果**：首页仪表盘自动刷新，显示关键指标
4. **AI咨询**：点击"AI对话中心"，询问健康问题
5. **趋势追踪**：在"健康档案"查看指标变化趋势图

### 演示数据
首次访问会自动创建演示数据：
- 默认成员：张伟（45岁，男）
- 演示报告：2024年体检报告
- 异常指标：尿酸 450 μmol/L、LDL-C 4.2 mmol/L
- AI建议：营养、运动、监测建议各1条

## 🛡️ 合规声明

**⚠️ 重要提示：**

本工具是健康信息参考工具，**不提供医疗诊断服务**。所有AI生成的建议、分析和解读仅供参考，不构成：
- ❌ 疾病诊断
- ❌ 治疗方案
- ❌ 用药建议
- ❌ 医疗咨询

**重要事项请咨询专业医师。紧急情况请拨打120。**
