# 智慧养老规划助手

一个面向35-65岁用户的轻量级、教育型养老自助规划工具，包含AI对话式信息采集和专业财务评估两大系统。

## 📋 项目简介

本项目提供两种养老规划方案：

### 🏠 方案一：生活规划助手
- **适合人群**：希望全面了解养老准备的用户
- **核心功能**：
  - 健康管理评估
  - 财务准备分析
  - 法律保障建议
  - 社会支持规划
- **交互方式**：AI对话式信息采集，自然流畅的问答体验

### 💰 方案二：财务准备度评估
- **适合人群**：关注养老财务安全的用户
- **核心功能**：
  - 6阶段渐进式对话评估
  - 精确的资金缺口计算
  - 可视化准备度仪表盘
  - 3步行动计划（短期/中期/长期）
- **特色**：专业的金融模型，考虑通胀、长寿风险等因素

## 🚀 技术栈

- **前端框架**：React 18 + TypeScript
- **路由**：React Router 7
- **UI组件**：Radix UI + Tailwind CSS v4
- **图标**：Lucide React
- **动画**：Motion (Framer Motion)
- **构建工具**：Vite 6

## 📦 安装与运行

### 前置要求
- Node.js 18+
- pnpm（推荐）或 npm

### 安装依赖
```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 启动开发服务器
```bash
# 使用 pnpm
pnpm dev

# 或使用 npm
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本
```bash
# 使用 pnpm
pnpm build

# 或使用 npm
npm run build
```

## 🎯 核心功能

### 生活规划助手特色
- ✅ AI智能助手"小智"引导式对话
- ✅ 6个维度渐进式信息采集
- ✅ 动态评分系统（0-100分）
- ✅ 彩色雷达图可视化展示
- ✅ 个性化行动计划生成
- ✅ 进度追踪与任务管理

### 财务评估系统特色
- ✅ 专业金融模型（通胀率3%、预期寿命85岁）
- ✅ 准备度仪表盘（状态分级：优秀/良好/需改善/亟待加强）
- ✅ 缺口来源分析（长寿风险45%+通胀30%+医疗25%）
- ✅ 智能金额解析（支持"2.5w"、"25000"等格式）
- ✅ 3步改善路径（本周可做/本季度重点/长期习惯）
- ✅ 端到端数据加密提示

## 🎨 设计特色

- **适老化设计**：大字体、高对比度、清晰按钮
- **主色调**：#4A90E2（专业蓝）+ 渐变色彩系统
- **响应式布局**：支持桌面端、平板、移动端
- **无障碍支持**：符合WCAG 2.1 AA标准

## 📝 使用说明

### 快速开始
1. 访问首页，选择"生活规划助手"或"财务准备度评估"
2. 与AI助手"小智"进行自然对话
3. 根据提示回答问题或选择快捷选项
4. 查看评估报告和行动建议
5. （可选）导出PDF报告或分享

### 数据隐私
- ✅ 所有数据存储在浏览器本地（LocalStorage）
- ✅ 不上传到任何服务器
- ✅ 可随时清除所有数据
- ✅ 退出后数据保留7天
